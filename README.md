# Weather-report-analysis 
Overview

This project analyzes historical weather data using Python to understand temperature patterns and other weather conditions. The notebook performs exploratory data analysis (EDA) to identify trends and relationships between different weather variables.

Dataset

The dataset weatherHistory.csv contains weather observations such as temperature, humidity, wind speed, visibility, pressure, and weather summary along with date and time information.

Tools & Libraries

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

Analysis

The notebook includes:

Loading and exploring the dataset

Data cleaning and preprocessing

Statistical summary of weather variables

Temperature trend analysis over time

Monthly aggregation to observe seasonal patterns

Conclusion

The analysis provides insights into weather trends and demonstrates how Python can be used for basic climate data exploration.
